# a node-vue todo-app

## App setup
```
cd > server
npm install
&
cd > client
npm install
```

### To start the server
```
cd > server
$ npm run serve
- mongo starts on http://localhost:4000/

```

### To start the client
```
cd > client
$ npm run serve

```

### To do tests
```
cd > server
$ npm run test

```

# Enjoy :)
